Educational Codeforces Round 187 (Rated for Div. 2)-CF-Edu-187-E. Probabilistic Card Game

Posted on 2026-02-25 Edited on 2026-02-26

题目大意

https://codeforces.com/contest/2203/problem/E

题目描述

Alice 和 Bob 有一个初始为空的牌堆。游戏共进行 mmm 回合，在第 iii 回合中，会发生以下事件：

  1. 一张数值为 aia_iai​ 的牌被加入牌堆（保证加入的牌数值各不相同）。

  2. 如果牌堆中的牌数少于 333 张，该回合直接结束。

  3. 否则，Alice 从牌堆中选择一张牌，设其数值为 aaa。

  4. 接着，Bob 在知道 Alice 选了哪张牌的情况下，从剩下的牌中选择一张牌，设其数值为 bbb（b≠ab \neq ab=a）。

  5. 最后，从剩下的 i−2i-2i−2 张牌中等概率随机抽取一张牌，设其数值为 ccc。

  6. 回合结束后，这三张牌被放回牌堆。

得分规则

在确定了 a,b,ca, b, ca,b,c 的值后，Bob 的得分按以下规则计算：

  • 如果 ∣a−c∣≤∣b−c∣|a - c| \le |b - c|∣a−c∣≤∣b−c∣，Bob 得 000 分；

  • 如果 ccc 的值严格介于 aaa 和 bbb 之间（即 a<c<ba < c < ba<c<b 或 b<c<ab < c < ab<c<a），Bob 得 000 分；

  • 其他情况下，Bob 得分为 ∣b−c∣|b - c|∣b−c∣。

游戏目标

在每回合中，Alice 的目标是最小化 Bob 的期望得分，而 Bob 的目标是最大化他的期望得分。两人均采取最优策略。（注意：两人是根据真实期望得分来做最小化或最大化的决策，而非取模后的值）。

输入格式

第一行包含一个整数 mmm（3≤m≤2⋅1053 \le m \le 2 \cdot 10^53≤m≤2⋅105），表示回合数。
第二行包含 mmm 个整数 a1,a2,…,ama_1, a_2, \dots, a_ma1​,a2​,…,am​（1≤ai≤10121 \le a_i \le 10^{12}1≤ai​≤1012，所有 aia_iai​ 互不相同），表示每回合加入牌堆的牌的数值。

输出格式

输出 m−2m-2m−2 个整数，第 iii 个数表示在第 i+2i+2i+2 回合中，双方在最优策略下 Bob 的期望得分对 998244353998244353998244353 取模后的结果。
