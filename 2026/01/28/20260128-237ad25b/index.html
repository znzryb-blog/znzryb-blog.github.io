题目大意

CF2169C Range Operation

题目描述

给定一个长度为 nnn 的整数数组 aaa。

你可以进行以下操作：选择一个区间 [l,r][l, r][l,r]（1≤l≤r≤n1 \le l \le r \le n1≤l≤r≤n），并将区间内的元素 al,al+1,…,ara_l, a_{l+1}, \dots, a_ral​,al+1​,…,ar​ 全部替换为 l+rl + rl+r。

你的任务是计算，在最多允许进行一次上述操作的情况下，数组的和可能达到的最大值。

输入格式

第一行包含一个整数 ttt（1≤t≤1041 \le t \le 10^41≤t≤104），表示测试用例的数量。

每个测试用例的第一行包含一个整数 nnn（1≤n≤2⋅1051 \le n \le 2 \cdot 10^51≤n≤2⋅105）。

第二行包含 nnn 个整数 a1,a2,…,ana_1, a_2, \dots, a_na1​,a2​,…,an​（0≤ai≤2n0 \le a_i \le 2n0≤ai​≤2n）。

输入额外限制：所有测试用例中 nnn 的总和不超过 2⋅1052 \cdot 10^52⋅105。

输出格式

对于每个测试用例，输出一个整数，表示在至多进行一次操作后数组的最大和。

思路讲解

类似于求解最大字段和的 dp。

思路参考了 starsilk 的代码。

首先，我们必须要把 r,lr,lr,l 两者分开。

我们就会发现，总答案就可以写为：（SxS_{x}Sx​ 表示 xxx 的前缀和）

ans=Sl−1+l(1−l)+Sn−Sr+r(r+1)ans=S_{l-1}+l(1-l)+S_n-S_{r}+r(r+1) ans=Sl−1​+l(1−l)+Sn​−Sr​+r(r+1)

这个明显可以使用这个 dp，因为我们只需要最优化 Sl−1+l(1−l)S_{l-1}+l(1-l)Sl−1​+l(1−l) 这个值，就直接最优化了左端点。而右端点的转移，一旦不用考虑这个左端点是不是最优的，就容易多了。

那我们定义，dpi,0dp_{i,0}dpi,0​ 为前缀和（操作前），dpi,1dp_{i,1}dpi,1​ 表示从 iii 及之前开始转化的最大化 Sl−1+l(1−l)S_{l-1}+l(1-l)Sl−1​+l(1−l) 值（可以理解为操作中，但是不严谨），dpi,2dp_{i,2}dpi,2​ 表示 iii 以及之前转化结束，所能达到的最大和（操作后）。

之所以定义三个状态，而不是两个，是为了避免重复操作。

dpi,0=dpi−1,0+aidpi,1=max⁡(dpi−1,1,dpi−1,0+i(1−i))dpi,2=max⁡(dpi−1,2+ai,dpi,1+i(i+1))dp_{i,0}=dp_{i-1,0}+a_i\\ dp_{i,1}=\max(dp_{i-1,1},dp_{i-1,0}+i(1-i))\\ dp_{i,2}=\max(dp_{i-1,2}+a_i,dp_{i,1}+i(i+1)) dpi,0​=dpi−1,0​+ai​dpi,1​=max(dpi−1,1​,dpi−1,0​+i(1−i))dpi,2​=max(dpi−1,2​+ai​,dpi,1​+i(i+1))

这个状态转移式子中 dpi,2dp_{i,2}dpi,2​ 之所以有两个来源，就是其状态定义，在 iii 之前转化结束一个来源，在 iii 结束一个来源。

iii 定义为这个长整型，小心溢出。
