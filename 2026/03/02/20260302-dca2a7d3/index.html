题目描述

给定一个长度为 nnn 的目标数组 TTT。

现有一个与 TTT 等长的初始数组 SSS，对 SSS 进行恰好一次如下操作：
选择一个整数 kkk，并将数组 SSS 划分为 kkk 个连续且不重叠的子段。假设这 kkk 个子段的下标区间依次为 [l1,r1],[l2,r2],…,[lk,rk][l_1, r_1], [l_2, r_2], \dots, [l_k, r_k][l1​,r1​],[l2​,r2​],…,[lk​,rk​]，满足 l1=1l_1=1l1​=1，rk=nr_k=nrk​=n，且对于所有的 1≤i≤k−11 \le i \le k-11≤i≤k−1，都有 ri+1=li+1r_i + 1 = l_{i+1}ri​+1=li+1​。
然后，对这 kkk 个子段分别独立地进行翻转操作。

请求出有多少种不同的初始数组 SSS，可以通过上述操作变为目标数组 TTT。由于答案可能很大，请输出其对 998244353998244353998244353 取模后的结果。

第二行包含 nnn 个整数 T1,T2,…,TnT_1, T_2, \dots, T_nT1​,T2​,…,Tn​（1≤Ti≤80001 \le T_i \le 80001≤Ti​≤8000），表示目标数组 TTT 的元素。
保证所有测试用例中 nnn 的总和不超过 800080008000。

输出格式

对于每组测试用例，输出一个整数，表示满足条件的数组 SSS 的数量对 998244353998244353998244353 取模后的结果。
