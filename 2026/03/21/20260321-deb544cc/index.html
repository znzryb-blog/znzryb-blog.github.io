2026 杭电春季联赛-1003-回文串

Posted on 2026-03-21

题目大意

题目描述
给定两个长度均为 nnn 的小写字母字符串 SSS 和 TTT。
你需要在 SSS 中选取一个子串 S′S'S′，在 TTT 中选取一个子串 T′T'T′（空串不算作子串），并将 T′T'T′ 拼接在 S′S'S′ 的后面组成一个新的字符串。
求共有多少种选取方式，使得拼接后的新字符串是一个回文串。
注：选取方式由子串在原串中的起始和终止位置决定；回文串指正读和反读都相同的字符串。

输入格式
第一行输入一个整数表示测试组数 TTT（T≤10T \le 10T≤10）。
接下来每组数据输入两行，分别表示字符串 SSS 和 TTT。
数据保证两个字符串长度相等且 n≤2000n \le 2000n≤2000。

输出格式
对于每组数据，输出一行整数，表示能组成回文串的子串选取方案数。

样例输入

样例输出

样例解释
对于第一组样例，S="ab"S = \text{"ab"}S="ab"，T="ac"T = \text{"ac"}T="ac"。合法的选取方式共 222 种：

  1. 选取 S[1…1]="a"S[1 \dots 1] = \text{"a"}S[1…1]="a" 和 T[1…1]="a"T[1 \dots 1] = \text{"a"}T[1…1]="a"，拼接得到 "aa"\text{"aa"}"aa"，是回文串。

  2. 选取 S[1…2]="ab"S[1 \dots 2] = \text{"ab"}S[1…2]="ab" 和 T[1…1]="a"T[1 \dots 1] = \text{"a"}T[1…1]="a"，拼接得到 "aba"\text{"aba"}"aba"，是回文串。

对于第二组样例，S="aba"S = \text{"aba"}S="aba"，T="bba"T = \text{"bba"}T="bba"。合法的选取方式共 151515 种，列举其中几种情况：

  • 选取 S[1…1]="a"S[1 \dots 1] = \text{"a"}S[1…1]="a" 和 T[3…3]="a"T[3 \dots 3] = \text{"a"}T[3…3]="a"，拼接得到 "aa"\text{"aa"}"aa"。

  • 选取 S[2…2]="b"S[2 \dots 2] = \text{"b"}S[2…2]="b" 和 T[1…1]="b"T[1 \dots 1] = \text{"b"}T[1…1]="b"，拼接得到 "bb"\text{"bb"}"bb"。

  • 选取 S[2…2]="b"S[2 \dots 2] = \text{"b"}S[2…2]="b" 和 T[2…2]="b"T[2 \dots 2] = \text{"b"}T[2…2]="b"，拼接得到 "bb"\text{"bb"}"bb"。

  • 选取 S[1…2]="ab"S[1 \dots 2] = \text{"ab"}S[1…2]="ab" 和 T[2…3]="ba"T[2 \dots 3] = \text{"ba"}T[2…3]="ba"，拼接得到 "abba"\text{"abba"}"abba"。

  • 选取 S[1…3]="aba"S[1 \dots 3] = \text{"aba"}S[1…3]="aba" 和 T[2…3]="ba"T[2 \dots 3] = \text{"ba"}T[2…3]="ba"，拼接得到 "ababa"\text{"ababa"}"ababa"。
    以此类推，满足条件的下标区间组合总共有 151515 种。
