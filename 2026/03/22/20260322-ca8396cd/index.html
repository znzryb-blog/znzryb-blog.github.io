Codeforces Round 1087 (Div. 2)——CF-2209-E. A Trivial String Problem（如何计算最短 border？）

Posted on 2026-03-22 Edited on 2026-03-23

题目大意

题目描述

定义函数 f(t)f(t)f(t) 为：将字符串 ttt 划分为若干部分的最大数量，满足每个部分都是 ttt 的非空前缀。

There exist kkk strings p1,p2,…,pkp_1,p_2,\ldots,p_kp1​,p2​,…,pk​, which are prefixes of ttt, such that t=p1+p2+…+pkt=p_1+p_2+\ldots+p_kt=p1​+p2​+…+pk​. Here +++ denotes the string concatenation.

换句话说，f(t)f(t)f(t) 是满足以下条件的最大正整数 kkk：存在 kkk 个字符串 p1,p2,…,pkp_1, p_2, \dots, p_kp1​,p2​,…,pk​，它们都是 ttt 的前缀，且 t=p1+p2+⋯+pkt = p_1 + p_2 + \dots + p_kt=p1​+p2​+⋯+pk​（+++ 表示字符串拼接）。

给定一个长度为 nnn 的小写英文字符串 sss。
你需要回答 qqq 次询问，每次询问给出两个整数 lil_ili​ 和 rir_iri​，要求计算并输出：
∑j=lirif(s[li…j])\sum_{j=l_i}^{r_i} f(s[l_i \dots j])∑j=li​ri​​f(s[li​…j])
（其中 s[li…j]s[l_i \dots j]s[li​…j] 表示 sss 从第 lil_ili​ 个字符到第 jjj 个字符组成的子串）。

这个其实是方便我们进行计算的。

输入格式

第一行包含一个整数 ttt（1≤t≤1031 \le t \le 10^31≤t≤103），表示测试用例的数量。
每个测试用例的第一行包含两个整数 nnn 和 qqq（1≤n≤1061 \le n \le 10^61≤n≤106, 1≤q≤1001 \le q \le 1001≤q≤100），分别表示字符串长度和询问次数。
第二行包含一个长度为 nnn 的小写英文字符串 sss。
接下来 qqq 行，每行包含两个整数 lil_ili​ 和 rir_iri​（1≤li≤ri≤n1 \le l_i \le r_i \le n1≤li​≤ri​≤n），表示一次询问。

保证所有测试用例中 nnn 的总和不超过 10610^6106。

输出格式

对于每次询问，输出一个整数表示表达式的求和结果。

样例数据

样例解释

在第一个测试用例中，f(a)=1f(\text{a}) = 1f(a)=1。

在第二个测试用例中：
对于第一次询问 1 5：f(a)+f(aa)+f(aaa)+f(aaaa)+f(aaaaa)=1+2+3+4+5=15f(\text{a}) + f(\text{aa}) + f(\text{aaa}) + f(\text{aaaa}) + f(\text{aaaaa}) = 1 + 2 + 3 + 4 + 5 = 15f(a)+f(aa)+f(aaa)+f(aaaa)+f(aaaaa)=1+2+3+4+5=15。
对于第二次询问 2 4（对应的子串是 aaa）：f(a)+f(aa)+f(aaa)=1+2+3=6f(\text{a}) + f(\text{aa}) + f(\text{aaa}) = 1 + 2 + 3 = 6f(a)+f(aa)+f(aaa)=1+2+3=6。

在第三个测试用例中：
对于第一次询问 1 6：f(a)+f(ab)+f(abc)+f(abcd)+f(abcde)+f(abcdef)=1+1+1+1+1+1=6f(\text{a}) + f(\text{ab}) + f(\text{abc}) + f(\text{abcd}) + f(\text{abcde}) + f(\text{abcdef}) = 1 + 1 + 1 + 1 + 1 + 1 = 6f(a)+f(ab)+f(abc)+f(abcd)+f(abcde)+f(abcdef)=1+1+1+1+1+1=6。
对于第二次询问 3 5（对应的子串是 cde）：f(c)+f(cd)+f(cde)=1+1+1=3f(\text{c}) + f(\text{cd}) + f(\text{cde}) = 1 + 1 + 1 = 3f(c)+f(cd)+f(cde)=1+1+1=3。
