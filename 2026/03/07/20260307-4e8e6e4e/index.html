题目大意

题目名称： Stone Steps

题目描述：
给定一个仅包含非零十进制数字的字符串 sss（长度 1≤∣s∣≤5×1051 \le |s| \le 5 \times 10^51≤∣s∣≤5×105）。

对于任意不包含数字 0 的正整数 nnn，定义 H(n)H(n)H(n) 为将 nnn 的所有数位按非降序（从小到大）重新排序后得到的新整数。例如：H(1971)=1179H(1971) = 1179H(1971)=1179，H(3)=3H(3) = 3H(3)=3。

你的任务是计算字符串 sss 的所有非空连续子串所代表的十进制整数的 HHH 函数值之和。即对于所有满足 1≤i≤j≤∣s∣1 \le i \le j \le |s|1≤i≤j≤∣s∣ 的子串 s[i...j]s[i...j]s[i...j]，求出 H(s[i...j])H(s[i...j])H(s[i...j]) 并求和。最终的结果需要对 1000696967 取模后输出。

输入格式：
单行输入一个字符串 sss。

输出格式：
输出一个整数，表示所有非空连续子串的 HHH 函数值之和对 1000696967 取模的结果。

样例 1 解释：
当 sss 为 3141 时，共有 10 个非空连续子串，计算过程如下：

  • s[1...1]s[1...1]s[1...1] 为 3，H(3)=3H(3) = 3H(3)=3

  • s[2...2]s[2...2]s[2...2] 为 1，H(1)=1H(1) = 1H(1)=1

  • s[3...3]s[3...3]s[3...3] 为 4，H(4)=4H(4) = 4H(4)=4

  • s[4...4]s[4...4]s[4...4] 为 1，H(1)=1H(1) = 1H(1)=1

  • s[1...2]s[1...2]s[1...2] 为 31，H(31)=13H(31) = 13H(31)=13

  • s[2...3]s[2...3]s[2...3] 为 14，H(14)=14H(14) = 14H(14)=14

  • s[3...4]s[3...4]s[3...4] 为 41，H(41)=14H(41) = 14H(41)=14

  • s[1...3]s[1...3]s[1...3] 为 314，H(314)=134H(314) = 134H(314)=134

  • s[2...4]s[2...4]s[2...4] 为 141，H(141)=114H(141) = 114H(141)=114

  • s[1...4]s[1...4]s[1...4] 为 3141，H(3141)=1134H(3141) = 1134H(3141)=1134

将其相加得：3+1+4+1+13+14+14+134+114+1134=14323 + 1 + 4 + 1 + 13 + 14 + 14 + 134 + 114 + 1134 = 14323+1+4+1+13+14+14+134+114+1134=1432。
