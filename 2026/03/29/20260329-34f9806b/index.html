2026 杭电春季联赛 2——1002-庭扫落樱（二分图最大匹配）

Posted on 2026-03-29 Edited on 2026-03-30

题目大意

题目描述

给定二维平面第一象限上的 nnn 个弹幕发射器。第 iii 个发射器位于坐标 (xi,yi)(x_i, y_i)(xi​,yi​)，并向四个方向之一发射一条射线（激光）：

  • di=0d_i = 0di​=0：向上发射

  • di=1d_i = 1di​=1：向右发射

  • di=2d_i = 2di​=2：向下发射

  • di=3d_i = 3di​=3：向左发射

射线的照明路径包含发射器本身所在的位置。题目保证任意两个发射器不会同时在对方的照明路径上。

现在可以在平面上放置障碍物（允许放置在发射器所在的坐标上），只要障碍物位于某条射线上，就能阻挡该射线的照射。求最少需要放置多少个障碍物，才能使得所有 nnn 个发射器的照射路径上都至少包含一个障碍物。

输入格式

第一行包含一个整数 ttt（1≤t≤1001 \le t \le 1001≤t≤100），表示测试用例的数量。

每个测试用例第一行包含一个整数 nnn（1≤n≤3001 \le n \le 3001≤n≤300），表示发射器的数量。
接下来 nnn 行，每行包含三个整数 xi,yi,dix_i, y_i, d_ixi​,yi​,di​（1≤xi,yi≤1091 \le x_i, y_i \le 10^91≤xi​,yi​≤109，di∈{0,1,2,3}d_i \in \{0, 1, 2, 3\}di​∈{0,1,2,3}），表示发射器的坐标与发射方向。

保证所有测试用例中 nnn 的总和不超过 300030003000。

输出格式

对于每个测试用例，输出一行一个整数，表示最少需要放置的障碍物数量。

样例解释

在第一个测试用例中，有 666 个发射器。一种最优方案是放置 333 个障碍物，具体位置如下：

  1. 在坐标 (5,7)(5, 7)(5,7) 处放置一个障碍物：该点覆盖了第 111 束（从 (5,5)(5,5)(5,5) 向上）、第 222 束（从 (3,7)(3,7)(3,7) 向右）以及第 555 束（从 (5,2)(5,2)(5,2) 向上）激光。

  2. 在坐标 (9,1)(9, 1)(9,1) 处放置一个障碍物：该点覆盖了第 333 束（从 (9,9)(9,9)(9,9) 向下）以及第 444 束（从 (9,1)(9,1)(9,1) 向左）激光。

  3. 在坐标 (−2025,20)(-2025, 20)(−2025,20) 处放置一个障碍物：该点覆盖了第 666 束（从 (1,20)(1,20)(1,20) 向左）激光。

该方案共使用了 333 个障碍物，且可以证明不存在比 333 更少的放置方案。
