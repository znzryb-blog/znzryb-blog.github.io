The 6th Liaoning Provincial Collegiate Programming Contest-2025-辽宁省赛-E. Entering the unknown（≤d 比=d 好计数更多，那为什么不使用 f(≤d)-f(≤d-1) 得到 f(=d) 呢？）

Posted on 2026-03-12 Edited on 2026-03-14

题目大意

题目描述
给定一个长度为 nnn 的正整数数组，求有多少个非空连续子数组满足以下条件：子数组内所有元素的和，能够被该子数组中所有元素包含的数字字符的最大值整除。

（注：数字字符指组成数字的 0∼90 \sim 90∼9。例如，数组 [2025, 11, 15] 中包含的数字字符有 0,1,2,50, 1, 2, 50,1,2,5，其最大值为 555。）

输入格式
第一行包含一个整数 TTT（1≤T≤1041 \le T \le 10^41≤T≤104），表示测试用例的数量。
对于每个测试用例：
第一行包含一个整数 nnn（1≤n≤1051 \le n \le 10^51≤n≤105），表示数组长度。
第二行包含 nnn 个正整数 x1,x2,…,xnx_1, x_2, \dots, x_nx1​,x2​,…,xn​（1≤xi≤1091 \le x_i \le 10^91≤xi​≤109），表示数组元素。
保证所有测试用例中 nnn 的总和不超过 10510^5105。

输出格式
对于每个测试用例，输出一行一个整数，表示满足条件的非空子数组数量。

样例输入

样例输出

样例解释
以第一个测试用例 [213, 12, 21] 为例，考察其所有非空子数组：

  • [213]：包含数字 1,2,31, 2, 31,2,3，最大数字为 333。子数组元素和为 213213213，213 mod 3=0213 \bmod 3 = 0213mod3=0，满足条件。

  • [12]：包含数字 1,21, 21,2，最大数字为 222。子数组元素和为 121212，12 mod 2=012 \bmod 2 = 012mod2=0，满足条件。

  • [21]：包含数字 1,21, 21,2，最大数字为 222。子数组元素和为 212121，21 mod 2≠021 \bmod 2 \neq 021mod2=0，不满足条件。

  • [213, 12]：包含数字 1,2,31, 2, 31,2,3，最大数字为 333。子数组元素和为 213+12=225213 + 12 = 225213+12=225，225 mod 3=0225 \bmod 3 = 0225mod3=0，满足条件。

  • [12, 21]：包含数字 1,21, 21,2，最大数字为 222。子数组元素和为 12+21=3312 + 21 = 3312+21=33，33 mod 2≠033 \bmod 2 \neq 033mod2=0，不满足条件。

  • [213, 12, 21]：包含数字 1,2,31, 2, 31,2,3，最大数字为 333。子数组元素和为 213+12+21=246213 + 12 + 21 = 246213+12+21=246，246 mod 3=0246 \bmod 3 = 0246mod3=0，满足条件。

满足条件的子数组共有 4 个。

思路讲解

≤d 比=d 好计数得多，那为什么不使用 f(≤d)-f(≤d-1) 得到 f(=d) 呢？

不过，要注意到，就是光一个 d 是不够描述这个 f 函数的。

定义 $ f(≤d，big_d)$ 指的是最大 digit 是 ≤d≤d≤d 而且可以被 bigdbig_dbigd​ 整除的这个子段数量。

因此，不难发现 $ f(=d，big_d)=f(≤d,big_d)-f(≤d-1,big_d)$。

使用这一容斥方法解决该问题，是因为我们发现在编程中，我们发现，很难对 最大 digit=d 进行计数（你要是不信的话可以自此试试）。
