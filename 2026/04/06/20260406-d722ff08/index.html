2026 杭电春季联赛 3——1009-时空回溯（向下取整除法的性质）

Posted on 2026-04-06 Edited on 2026-04-07

题目大意

题目描述

给定一个长度为 nnn 的数组 AAA 以及一个阈值 mmm。你需要维护这个数组并支持以下两种操作，共执行 QQQ 次：

  • 1 L R X：将区间 [L,R][L, R][L,R] 内的所有元素减去 XXX（即对于 L≤i≤RL \le i \le RL≤i≤R，执行 Ai←Ai−XA_i \leftarrow A_i - XAi​←Ai​−X）。

  • 2 L R：将区间 [L,R][L, R][L,R] 内的所有元素右移一位（相当于除以 222 向下取整，即对于 L≤i≤RL \le i \le RL≤i≤R，执行 Ai←⌊Ai2⌋A_i \leftarrow \lfloor \frac{A_i}{2} \rfloorAi​←⌊2Ai​​⌋）。

在每一次操作结束后，如果整个数组中存在至少一个元素小于等于阈值 mmm，则会触发一次“时空回溯”：

  1. 全局重置次数 kkk 的值增加 111。

  2. 整个数组 AAA 立即恢复为最开始的初始状态。

要求在处理完所有给定的 QQQ 次操作后，输出最终的重置次数 kkk（初始时 k=0k=0k=0）。

数据范围

  • 数据组数 T≤10T \le 10T≤10。

  • 1≤n,Q≤1051 \le n, Q \le 10^51≤n,Q≤105。

  • 0≤m<2300 \le m < 2^{30}0≤m<230。

  • 初始数组满足 m<Ai<230m < A_i < 2^{30}m<Ai​<230。

  • 对于操作 1：1≤L≤R≤n1 \le L \le R \le n1≤L≤R≤n，0≤X<2300 \le X < 2^{30}0≤X<230。

  • 对于操作 2：1≤L≤R≤n1 \le L \le R \le n1≤L≤R≤n。

样例输出

样例解释

初始时数组为 [5,4,6][5, 4, 6][5,4,6]，阈值 m=1m = 1m=1，重置次数 k=0k = 0k=0。

  • 第 111 次操作（1 1 2 4）：对区间 [1,2][1, 2][1,2] 的元素减去 444，数组变为 [1,0,6][1, 0, 6][1,0,6]。此时存在元素（111 和 000）小于等于阈值 111，触发回溯。重置次数 k←1k \leftarrow 1k←1，数组重置为初始状态 [5,4,6][5, 4, 6][5,4,6]。

  • 第 222 次操作（2 2 3）：对区间 [2,3][2, 3][2,3] 的元素右移一位，数组变为 [5,2,3][5, 2, 3][5,2,3]。此时数组中所有元素均大于 111，不触发回溯。

  • 第 333 次操作（1 1 3 5）：对区间 [1,3][1, 3][1,3] 的元素减去 555，数组变为 [0,−3,−2][0, -3, -2][0,−3,−2]。此时存在元素小于等于阈值 111，触发回溯。重置次数 k←2k \leftarrow 2k←2，数组重置为初始状态 [5,4,6][5, 4, 6][5,4,6]。

所有操作执行完毕，最终的重置次数 kkk 为 222。
