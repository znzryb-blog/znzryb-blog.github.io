2026 杭电春季联赛 4——1005-列车停放站

Posted on 2026-04-12 Edited on 2026-04-13

题目大意

题目描述
给定 nnn 辆列车的停放区间 [l,r][l, r][l,r]。
有 mmm 次询问，每次询问给出一个空区间 [x,y][x, y][x,y]，需要求出在该空区间内最多可以完整停放多少辆列车。
被选择停放的列车区间必须完全包含在询问区间 [x,y][x, y][x,y] 内，且不同列车的停放区间在内部不能重叠，但端点可以相交（即列车之间可以紧贴，不留空隙）。

输入格式
第一行输入一个整数 TTT（1≤T≤51 \le T \le 51≤T≤5），表示测试数据组数。
对于每组数据：
第一行输入一个整数 nnn（1≤n≤1051 \le n \le 10^51≤n≤105），表示列车数量。
接下来 nnn 行，每行输入两个整数 l,rl, rl,r（1≤l<r≤1091 \le l < r \le 10^91≤l<r≤109），表示一辆列车的停放区间为 [l,r][l, r][l,r]。
接下来一行输入一个整数 mmm（1≤m≤1051 \le m \le 10^51≤m≤105），表示询问次数。
接下来 mmm 行，每行输入两个整数 x,yx, yx,y（1≤x<y≤1091 \le x < y \le 10^91≤x<y≤109），表示每次询问的空区间。

输出格式
对于每次询问，输出一行一个整数，表示最多可以在该区间内停放的列车数量。

样例输入

样例输出

样例解释
在该样例中，共有 333 辆列车，区间分别为 [1,3][1, 3][1,3]、[1,4][1, 4][1,4]、[3,4][3, 4][3,4]。

  • 第一次询问区间 [1,3][1, 3][1,3]：只有第 111 辆列车 [1,3][1, 3][1,3] 能够完整停放在该区间内，因此最多停放 111 辆列车。

  • 第二次询问区间 [1,4][1, 4][1,4]：可以选择第 111 辆列车 [1,3][1, 3][1,3] 和第 333 辆列车 [3,4][3, 4][3,4]。这两辆列车在端点 333 处紧贴，且都完整包含在询问区间 [1,4][1, 4][1,4] 内，不发生重叠，因此最多停放 222 辆列车。
