三年趋势（v3 重算，作废 v2 旧数字）

v2 旧数字（92% 场次有几何）作废：那是把防 AK + 金档偏难也算进去了，对 zzy 队伍没训练价值。v3 按「大部分金牌队过 OR 大部分银牌队过」硬口径重算，只保留 🥇 金 ∪ 🥈 银 ∪ 🥉 铜 = 12 道（7 + 1 + 4），剔除金档偏难 / 接近防 AK / 防 AK。

口径：仅算 Regional + EC Final + CCPC 站赛 + CCPC Final（38 场），剔线上 / 高职 / 女生赛（无 srk / 不可比）。

关键结论（与 v2 完全相反）

  1. 超过半数比赛根本没有 zzy 能做的几何题：2023 有 77% 场次 0 道、2024 有 69%、2025 有 58%。「每场都要面对几何题」是 v2 错觉——大多数场次的几何题是金牌队都做不出来的防 AK，跟 zzy 队伍训练强度无关。

  2. 趋势仍上升但不夸张：从 23% → 42%（不是 v2 说的 85% → 92%）。三年翻不到 2 倍。

  3. 2025 拐点：5/12 场有可做几何，比例首次破 1/3。继续涨的话 2026 可能到 50%。

  4. 每场 0.42 道是什么概念：12 场比赛大概只 5 道几何 zzy 能稳吃——比「金/银题里的 DS / DP / 图论」少一个量级。

  • 2024（4 道）：Concave Hull (CCPC 哈尔滨 B, 金) · Staircase Museum (ICPC EC Final H, 金) · Horizon Scanning (ICPC 昆明 H, 铜签到) · Guess the Polygon (ICPC 沈阳 G, 金)

  • 2023（3 道）：Painter (CCPC 哈尔滨 M, 铜签到) · Almost Convex (ICPC 济南 M, 铜签到) · Three Rectangles (ICPC 沈阳 I, 金)

对队伍策略的修正

  • 不需要培养专门的「几何手」：3 年只 12 道可做几何，平均每场不到半道。培养代价 vs 期望收益不成比例。

  • 应当掌握的几何子套路（覆盖 12 道里的核心）：凸包 + 凸多边形面积 / 重心 / 闵可夫斯基（占 7-8 道）+ 极角扫描（Horizon Scanning 类）+ 矩形覆盖布尔（Three Rectangles / Outstanding Outlines）。不需要练 3D / visibility / 半平面布尔表达式 / 反射 / 同心圆门 / Manhattan 期望——这些题虽然每年都有，但全在防 AK / 偏难，目标段位（金/银）做不出来也没事。

  • 真正的策略价值：拿到比赛后先扫一眼有没有 Tier 金/银的几何题，有就值得花时间想；没有就果断放弃几何题（58% 概率没有），把时间留给 DS / DP。

假阳性踩坑

按「赛站名 + 年份」fuzzy match Gym 镜像很容易混淆城市/学校重名：

  • Xi'an 子串混进 **Xian**gtan University / **Xian** Jiaotong University → 湘潭 PC + 西交大校赛被误命中为 Xi’an Regional

  • Shanghai 混进 **Shanghai** Collegiate Programming Contest（上海高校赛）→ 误命中为 ICPC Asia Shanghai Regional

修法：matcher 必须加 \b(ICPC.*Asia|Universal.*Cup)\b 限定，把校赛 / 高校赛 / 邀请赛过滤掉。本轮 4 个 FP 已手动剔除并删除对应 PDF。
