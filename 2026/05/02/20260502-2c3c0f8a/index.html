思路讲解

一句话

把「区间异或为 0」翻译成「两个端点的前缀异或相等」，再用 f(n)=0⊕1⊕⋯⊕nf(n) = 0 \oplus 1 \oplus \dots \oplus nf(n)=0⊕1⊕⋯⊕n 的 mod 4 周期把「f 等于啥」压成 4 类，O(1) 计数。

第 1 步：XOR 前缀化

记 f(n)=0⊕1⊕⋯⊕n\textcolor{blue}{\boldsymbol{f(n) = 0 \oplus 1 \oplus \dots \oplus n}}f(n)=0⊕1⊕⋯⊕n （约定 f(−1)=0f(-1) = 0f(−1)=0 ）。那么

l⊕(l+1)⊕⋯⊕r=f(r)⊕f(l−1)l \oplus (l + 1) \oplus \dots \oplus r = f(r) \oplus f(l - 1) l⊕(l+1)⊕⋯⊕r=f(r)⊕f(l−1)

区间异或为 0 ⟺ f(r)=f(l−1)f(r) = f(l - 1)f(r)=f(l−1) 。

为了少写一个 −1-1−1 ，把左端点替身记成 m=l−1\textcolor{blue}{\boldsymbol{m = l - 1}}m=l−1 。原约束 1≤l≤x≤r≤n1 \le l \le x \le r \le n1≤l≤x≤r≤n 翻译成：

  • 0≤m≤x−10 \le m \le x - 10≤m≤x−1

  • x≤r≤nx \le r \le nx≤r≤n

  • f(m)=f(r)f(m) = f(r)f(m)=f(r)

注意 m 严格小于 r （ m≤x−1<x≤rm \le x - 1 < x \le rm≤x−1<x≤r ），后面用得上。

第 3 步：哪些 (m, r) 配对能成立

关键观察： m<rm < rm<r 严格成立。所以「f(m) = f® 且这个值还依赖 m / m+1 / r / r+1 这种大值」的情形，逼出来都要求 m=rm = rm=r 之类等式才能配，被严格不等卡死。
