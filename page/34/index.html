P3379 【模板】最近公共祖先（LCA）（树链剖分做法）

Posted on 2025-07-21 Edited on 2026-02-17

题目大意

给定一棵有根多叉树，包含 NNN 个节点，树根为节点 SSS。有 MMM 次询问，每次询问给定两个节点 aaa 和 bbb，求它们的最近公共祖先（LCA，Lowest Common Ancestor）。

输入格式：

  • 第一行：NNN MMM SSS（节点数、询问数、根节点）

  • 接下来 N−1N-1N−1 行：每行两个整数 xxx yyy，表示 xxx 和 yyy 之间有一条边

  • 接下来 MMM 行：每行两个整数 aaa bbb，表示询问 aaa 和 bbb 的最近公共祖先

输出格式：

  • MMM 行，每行一个整数，表示对应询问的答案

思路讲解

重点在于这两个dfs，其实还挺简单的。dfs负责找出重儿子，dfss负责重链剖分。

Posted on 2025-07-20 Edited on 2026-02-17

思路讲解

三个集合的容斥原理

$\left | A \cup B \cup C \right | = \left | A \right | + \left | B \right | + \left | C \right | - \left | A \cap B \right | - \left | A \cap C \right | - \left | B \cap C \right | + \left | A \cap B \cap C \right | $

那么不难发现规律，就是奇加偶减。这就是这段代码的由来。

那么直接计算合法路径数量比较难，但是我们可以枚举不合法子集。如果s是101（二进制），那么就是第一个素因子要不符合要求（LCM只关注这些数当中的最大的，你可以理解为对素因子取了一个并集，那么gcd就是取了一个交集）。
