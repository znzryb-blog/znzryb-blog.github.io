相关定义

  • kkk-松散子序列：如果子序列 TTT 在原字符串 SSS 中对应的下标序列为 pos1,pos2,…,posmpos_1, pos_2, \dots, pos_mpos1​,pos2​,…,posm​，且满足对于所有的 i∈[1,m−1]i \in [1, m-1]i∈[1,m−1]，都有 posi+1−posi>kpos_{i+1} - pos_i > kposi+1​−posi​>k，那么 TTT 就是一个 kkk-松散子序列。即子序列中原串相邻提取字符的原始位置距离必须严格大于 kkk。

题目大意

题目描述
给定一个长度为 nnn 的数组 aaa 和一个参数 kkk。如果一个数组 bbb 满足以下条件，则称其为“很酷的”：
对于从 kkk 到 nnn 的每个下标 iii，子数组 [ai−k+1,ai−k+2,…,ai][a_{i-k+1}, a_{i-k+2}, \dots, a_i][ai−k+1​,ai−k+2​,…,ai​] 都是子数组 [bi−k+1,bi−k+2,…,bi][b_{i-k+1}, b_{i-k+2}, \dots, b_i][bi−k+1​,bi−k+2​,…,bi​] 的重新排列（即两个长度为 kkk 的滑动窗口构成的多重集相同）。

现在给定数组 aaa 和数组 bbb。数组 aaa 仅包含 111 到 nnn 之间的整数；数组 bbb 包含 111 到 nnn 之间的整数以及 −1-1−1。
请判断是否可以通过将 bbb 中所有的 −1-1−1 替换为 111 到 nnn 之间的任意整数，使得最终的数组 bbb 满足上述“很酷的”条件。

输入格式
第一行包含一个整数 ttt（1≤t≤1041 \le t \le 10^41≤t≤104），表示测试用例的数量。
每个测试用例的第一行包含两个整数 nnn 和 kkk（1≤k≤n≤2⋅1051 \le k \le n \le 2 \cdot 10^51≤k≤n≤2⋅105）。
第二行包含 nnn 个整数 a1,a2,…,ana_1, a_2, \dots, a_na1​,a2​,…,an​（1≤ai≤n1 \le a_i \le n1≤ai​≤n）。
第三行包含 nnn 个整数 b1,b2,…,bnb_1, b_2, \dots, b_nb1​,b2​,…,bn​（1≤bi≤n1 \le b_i \le n1≤bi​≤n 或 bi=−1b_i = -1bi​=−1）。
保证所有测试用例中 nnn 的总和不超过 2⋅1052 \cdot 10^52⋅105。

输出格式
对于每个测试用例，如果可以构造出满足条件的数组 bbb，输出 YES，否则输出 NO。

样例

样例解释第一个测试用例：k=5k=5k=5。唯一的长度为 555 的子数组是整个数组本身。给定的 bbb 已经是 aaa 的一个重新排列，且没有 −1-1−1，满足条件，答案为 YES。
第二个测试用例：k=2k=2k=2。我们可以将所有的 −1-1−1 替换，使得 b=[2,1,2,1,2]b = [2, 1, 2, 1, 2]b=[2,1,2,1,2]。此时在 aaa 和 bbb 中，每一个长度为 222 的窗口要么是 [1,2][1, 2][1,2] 要么是 [2,1][2, 1][2,1]，它们互为重新排列，因此答案为 YES。
