题目设定
给定一个包含 kkk（1≤k≤261 \le k \le 261≤k≤26）首短歌的曲库，分别用前 kkk 个大写英文字母表示。
由这 kkk 首歌可以生成一个无限长的母带（master track）ttt。生成方式为：不断重复这 kkk 首歌，但在每一次重复整个曲库时，都会将当前序列的第一首歌移动到序列的末尾。
例如，当 k=4k = 4k=4 时，曲库初始为 ABCD，那么母带 ttt 就是无限长的字符串 ABCD BCDA CDAB DABC ABCD ...。

有一个长度为 nnn 的播放列表，初始时全部为空白。在此问题中，播放列表和母带均从 1 开始索引。

操作要求
需要对播放列表执行 qqq 次操作，操作分为以下两种：

  1. + i a b：将播放列表中从位置 aaa 到 bbb（包含两端）的片段，替换为母带 ttt 中从位置 iii 开始、长度为 b−a+1b - a + 1b−a+1 的子串。

  2. ? a b：查询播放列表中从位置 aaa 到 bbb（包含两端）的片段。输出 kkk 个整数，其中第 jjj 个整数表示该片段中第 jjj 个英文字母（即第 jjj 首歌）的出现次数。

输入格式
第一行包含三个空格分隔的整数 kkk、nnn 和 qqq。
接下来 qqq 行，每行描述一个操作，格式为 + i a b 或 ? a b。

输出格式
对于每一个 ? a b 操作，输出一行包含 kkk 个空格分隔的整数，表示查询的结果。

数据范围
1≤k≤261 \le k \le 261≤k≤26
1≤n≤1091 \le n \le 10^91≤n≤109
1≤q≤1051 \le q \le 10^51≤q≤105
每次操作中：1≤i≤1091 \le i \le 10^91≤i≤109，1≤a≤b≤n1 \le a \le b \le n1≤a≤b≤n
